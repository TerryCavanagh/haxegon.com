Haxegon

A programming library for beginners

Currently v0.12.1

About

Haxegon is a tiny Haxe programming library aimed at beginners. It is powered by OpenFL and Starling.

Haxegon compiles natively to HTML5 and Desktop (Windows, MacOS, Linux, Neko). Flash is also supported as a legacy/development platform.

It is currently in alpha, so things are changing fast and stuff might be broken. You can check out the source on GitHub.

Thanks for checking it out! If you have any feedback, you can email the Haxegon team at helloeveryone [at] haxegon [dot] com or you can contact the author of Hexagon on Twitter.
