Data

Simple loading and saving of data, and reading in textfiles and CSV files.
Data.loadtext(textfile:String):Array
Loads in a .txt file from data/text/ as an array of Strings.
Data.loadcsv(csvfile:String, delimiter = ","):Array<???>
Loads in a .csv file from data/text/ as an array of values.
Data.load2dcsv(csvfile:String, delimiter = ","):Array<?>
Loads in a two-dimensional .csv file from data/text/ as a 2d array of values. Use Data.width and Data.height to read the width and height of the loaded 2d CSV.

Helpful function to give you a blank 2D array of given width and height!
Data.savefile:String
Optional! Sets the name of the savefile to that Data.load() and Data.save() will use.
Data.save(key:String, value:?)
Saves a value. Can be loaded later with Data.load().
Data.load(key:String)
Loads a value that you've previously saved with Data.save(). If no saved value for that key exists, then it returns 0.
