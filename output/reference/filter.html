Filter

Use simple fullscreen filters.
Filter.bloom:Float
Turns on a fullscreen bloom filter. You can set it to 1.0 for a normal bloom effect, but any value will work.
Filter.blur:Bool
Turns on a fullscreen blur filter. Set to true or false to toggle on and off.
Filter.reset()
Turns off all currently enabled filters.
