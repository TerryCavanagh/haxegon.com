Random

Pick things randomly.
Random.int(from:Int, to_inclusive:Int):Int
Returns an Integer between 'from' and 'to_inclusive'.
Random.float(from:Int, to_inclusive:Int):Float
Returns a Float between 'from' and 'to_inclusive'.
Random.string(length:Int, ?characters:String):String
Returns a random String of a given length. You can optionally specify which characters to use, otherwise the default is (a-zA-Z0-9).
Random.random():Float
Returns an Float between 0 and 1.
Random.bool():Bool
50:50 chance of being true/false
Random.chance(n:Float):Bool
n% chance (out of 100%) of being true.
