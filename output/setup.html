Rough guide:

1 - Install the Haxe compiler on your machine: Haxe.org

2 - Install the OpenFL library: OpenFl - Getting Started

3 - Install the Haxegon library: haxelib install haxegon

4 - Install an IDE (for windows try HaxeDevelop, for Mac/Linux try Sublime Text 2).

5 - Download a blank haxegon project!

You can find examples here: haxegon-samples

And full reference documentation here: Reference
